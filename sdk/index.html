Quick

Drop a folder of HTML, get a site. Deploy from any directory:

quick deploy mysite

Your site is live at https://mysite.<quick-domain>/ and every page can use the zero-config backend:

<script src="/quick.js"></script>
<script>
  const posts = quick.db.collection('posts');
  await posts.create({ title: 'Hello Quick DB' });
  const me = await quick.me(); // { email, name, peer_name, peer_ip }
</script>

Run quick init in a project to teach your coding agent the whole API.
