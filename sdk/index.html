Spot

Drop a folder, get a spot. Deploy from any directory:

spot deploy mysite

Your site is live at https://mysite.<spot-domain>/ and every page can use the zero-config backend:

<script src="/spot.js"></script>
<script>
  const posts = spot.db.collection('posts');
  await posts.create({ title: 'Hello Spot DB' });
  const me = await spot.me(); // { email, name, peer_name, peer_ip }
</script>

Run spot init in a project to teach your coding agent the whole API.
